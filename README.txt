# draft_proposal
Draft Proposal Ilmu Komputer UNUD

Ketentuan:
1. Tiap folder dinamai dengan NIM dari masing-masing mahasiswa.
2. Tiap folder berisi dokumen draft proposal dari masing-masing mahasiswa yang diberi nama "proposal_<<NIM>>.doc(x)"
3. Dokumen dapat menggunakan format .doc, .docx, .rtf, .ps, atau .tex
